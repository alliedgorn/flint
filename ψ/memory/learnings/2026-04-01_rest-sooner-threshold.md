# Lesson: Rest-Sooner Threshold Works

**Date**: 2026-04-01
**Source**: Sessions 52-56, five consecutive empty watches

## Pattern

Three empty board checks is the right trigger for rest-sooner. Don't extend the watch hoping work will appear — if the board is empty and Gorn is silent, rest quickly. Five sessions confirmed this: only S52 had real work, the other four were pure maintenance overhead.

## Corollary

Push to origin before resting if ahead. Don't let commits accumulate across rest cycles — it creates ambiguity about repo state for other Beasts.
