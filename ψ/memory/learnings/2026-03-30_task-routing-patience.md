**Date**: 2026-03-30
**Context**: Session 14 — T#492 called for Flint by Zaghnal and Leonard but not formally assigned

1. Formal assignment in the board system (assignee field set)

When a task is mentioned with your name but the board still shows `assignee: None`. Ask Gorn, then wait.
